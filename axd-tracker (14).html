🔄 Cloud Sync (Supabase) — sync across all devices
Setup in 5 minutes — completely free:
1. Go to supabase.com → sign up → New project
2. Once created, go to Project Settings → API
3. Copy Project URL and anon public key
4. Go to SQL Editor and run this query once:
CREATE TABLE axd_data (id TEXT PRIMARY KEY, data JSONB, updated_at TIMESTAMPTZ DEFAULT NOW()); 5. Paste your URL and key below and click Save
⚙️ EmailJS setup required — follow these steps to enable email notifications:
  1. Go to emailjs.com → create a free account
  2. Add an Email Service (Gmail recommended) → copy your Service ID
  3. Create an Email Template with variables: {{to_name}}, {{message}}, {{from_name}} → copy Template ID
  4. Go to Account → copy your Public Key
  5. Paste all three below and click Save
